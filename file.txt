Hello

git -> version control
github -> storage

commands
------------------

git init -> initialize

git config --global user.name "mern-trainer"
git config --global user.email "email"

git add . -> stage all files
git add <file1> <file2> -> spec. file

git commit -m "<message>"

git remote add <remote_name> <remote_url> -> set remote

git config --get remote.<remote_name>.url -> check remote url

git push <remote_name> <branch_name> -> push changes to github


branches
----------------------------------------

git branch -> list of branches
git branch <branch_name> -> create a new branch
